HOW TO RUN THE PORTFOLIO WEBSITE
================================

This portfolio is built using React, Vite, and Tailwind CSS. Follow these steps to run it locally on your machine using the terminal.

Prerequisites:
--------------
Make sure you have Node.js and npm installed on your system.
You can download Node.js from: https://nodejs.org/

Steps to Run:
-------------

1. Open your Terminal (PowerShell, Command Prompt, or terminal of your choice).

2. Navigate to the project directory:
   cd "C:\Users\sachi\OneDrive\Desktop\Portfolio"

3. Install dependencies (only required the first time):
   npm install

4. Start the local development server:
   
   * Using Windows Command Prompt (CMD):
     npm run dev
     
   * Using Windows PowerShell (if you run into execution policy errors):
     cmd /c npm run dev
     
5. Open your web browser and navigate to the address shown in the terminal (usually http://localhost:5173/).

Other Commands:
---------------
* To build the project for production:
  npm run build

* To preview the production build locally:
  npm run preview
